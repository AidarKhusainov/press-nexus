# ADR Index

Architecture Decision Records capture significant architecture decisions.

## How To Work With ADRs

- Create a new document for each significant decision.
- Filename format: `NNNN-short-title.md`.
- Statuses: `Proposed`, `Accepted`, `Superseded`, `Deprecated`.
- Any change to an existing decision must be recorded as a new ADR, not by rewriting history.
